## Why

현재 앱은 알파벳/단어/대화를 **보고 듣기만** 하는 수동적 학습 방식이다. 아이들의 흥미를 유발하고 학습 효과를 높이려면 **직접 누르고, 옮기고, 반응하는 인터랙티브 활동**이 필요하다. 퀴즈와 놀이 카테고리를 추가하여 아이들이 능동적으로 참여할 수 있는 6가지 활동을 제공한다.

## What Changes

### 카테고리 선택 화면
- **퀴즈(🧩)** 버튼 추가: 선택 시 퀴즈 유형 선택 화면으로 이동
- **놀이(🎮)** 버튼 추가: 선택 시 놀이 유형 선택 화면으로 이동
- `LearningCategory` 타입에 `'quiz' | 'play'` 추가

### 퀴즈 카테고리 (4가지 유형)

**1. 소리 듣고 고르기** 🔊
- 퀴즈 유형 선택 → 주제 선택 (알파벳/음식/요리/동물/탈것/몸)
- 상단: '소리를 듣고 맞는 것을 고르세요'
- 중앙: 물음표(❓) 카드 → 정답 시 이모지로 전환
- 하단: 3개의 선택지 (이모지 + 텍스트)
- 음성 2번 자동 재생 후 선택, 정답 시 효과음 + 다음 문제, 오답 시 소리 재재생
- 5문제 후 결과 화면

**2. 그림 보고 단어 고르기** 🖼️
- 중앙: 이모지/그림 카드 표시
- 하단: 3개의 영어 단어 선택지
- 그림을 보고 맞는 영어 단어를 고르기
- 정답 시 발음 재생 + 효과음, 오답 시 해당 단어 발음 재생
- 5문제 후 결과 화면

**3. 첫 소리 맞추기** 🔤
- 중앙: 이모지/그림 카드 + 단어 표시
- 하단: 3개의 알파벳 선택지
- '이 단어는 어떤 소리로 시작할까?' 안내
- 정답 시 알파벳 발음 재생 + 효과음, 오답 시 해당 알파벳 발음 재생
- 5문제 후 결과 화면

**4. 철자 맞추기** ✏️
- 중앙: 이모지/그림 + 빈칸이 포함된 단어 (예: A _ _ l e)
- 하단: 알파벳 글자 선택지 (4~5개)
- 빈칸에 들어갈 알맞은 글자를 선택
- 정답 시 효과음 + 빈칸 채워짐, 오답 시 흔들림 애니메이션
- 5문제 후 결과 화면

### 놀이 카테고리 (2가지 유형)

**5. 매칭 게임 (카드 뒤집기)** 🃏
- 주제 선택 (알파벳/음식/요리/동물/탈것/몸)
- 뒤집힌 카드 그리드 (4x3 = 6쌍)
- 카드를 2장 뒤집어 짝 맞추기 (이모지 ↔ 영어 단어, 또는 대문자 ↔ 소문자)
- 짝이 맞으면 카드 유지 + 효과음, 틀리면 다시 뒤집힘
- 모든 짝을 맞추면 완료 (시도 횟수 표시)

**6. 드래그 앤 드롭 분류** 🎯
- 2개의 카테고리 바구니 표시 (예: 동물 🐾 / 음식 🍽️)
- 화면에 섞인 단어 카드들이 나타남
- 카드를 알맞은 바구니로 끌어다 놓기
- 정답 시 효과음 + 카드 흡수 애니메이션, 오답 시 카드 되돌아감
- 모든 카드 분류 완료 시 결과 화면

### 공통
- **결과 화면**: 정답 수/총 문제 수, 별점(★) 표시, '다시 하기' / '뒤로' 버튼
- **효과음**: 정답/오답/완료 효과음 추가

## Capabilities

### New Capabilities
- `quiz-mode`: 4가지 퀴즈 유형 (소리 듣고 고르기, 그림 보고 단어 고르기, 첫 소리 맞추기, 철자 맞추기) — 유형 선택 → 주제 선택 → 5문제 풀기 → 결과 화면
- `play-mode`: 2가지 놀이 유형 (매칭 게임, 드래그 앤 드롭 분류) — 유형 선택 → 주제 선택 → 플레이 → 결과 화면

## Impact

- **타입 변경**: `LearningCategory`에 `'quiz' | 'play'` 추가, 퀴즈/놀이 유형 타입 정의 (`src/types.ts`)
- **라우팅 추가**: `/quiz-types`, `/quiz/:type/:category`, `/play-types`, `/play/:type/:category` 등 (`src/routes.tsx`)
- **카테고리 선택 수정**: `CategorySelection.tsx`에 퀴즈/놀이 버튼 추가
- **신규 컴포넌트**:
  - 퀴즈: `QuizTypes.tsx`, `QuizCategories.tsx`, `QuizListenAndChoose.tsx`, `QuizImageToWord.tsx`, `QuizFirstSound.tsx`, `QuizSpelling.tsx`, `QuizResult.tsx`
  - 놀이: `PlayTypes.tsx`, `PlayCategories.tsx`, `PlayMatchingGame.tsx`, `PlayDragAndDrop.tsx`, `PlayResult.tsx`
- **기존 데이터 활용**: `alphabets.json`, `vocabulary.json`의 오디오/이모지/스펠링 데이터를 퀴즈·놀이에서 재사용
- **효과음**: 정답/오답/완료 효과음 파일 추가 (`public/assets/audio/`)
- **드래그 앤 드롭**: HTML5 Drag and Drop API 또는 터치 이벤트 처리 (추가 라이브러리 검토 필요)
- **기타 의존성**: 기본적으로 추가 라이브러리 불필요 (DnD는 설계 단계에서 결정)
